Server configuration:

Players

The maximum number of players that can play on the server at the same time.

If non-zero players will be kicked for being idle for that many minutes.

Server

Changes the port the server is hosting on.

Enable command block

Enables command blocks

The maximum number of milliseconds a single tick may take before the server watchdog stops the server

Allow flight

Allows users to use flight on your server while in Survival mode. (bumbo: enabling can lead to jump-sleeping bug)

Sets the default permission level for ops when using /op command.

Enable whitelist

Players that are not in whitelist cannot log into the server.

Enforce whitelist

Determines whether players who are not present in the whitelist will be kicked or not.

Broadcast console to ops

Whether to broadcast console to ops*

Snooper Enabled

Enable snooper that sends data to http://snoop.minecraft.net.

Use native transport

Enable linux server optimizations.

Prevent proxy connections

If player uses proxy connection, he is kicked.

RCON

Enable rcon

Enables remote access to server console

Broadcast rcon to ops

Whether to broadcast rcon to ops*

Sets the port to RCON.

Sets the password to RCON.

Gameplay

Difficulty

Defines the difficulty of the server.

Enable PVP

Enable PVP on the server.

Force gamemode

Force players to join the the default gamemode.

Gamemode

Defines the mode of gameplay.

Hardcore

If true, difficulty is ignored and set to hard and player will be set to spectator mode if they die.

Spawn

Spawn NPCs

Determines whether villagers will be spawned.

Spawn monsters

Determines if the monsters are going to be spawned.

World

Allow nether

Allows players travel to nether.

Determines the radius of the spawn protection.

Maximum amound of chunks that can be loaded by any player. (mojang: Value less than 9 causes MC-2536 bug.)

Spawn animals

Determines whether animals will be spawned on the server.

This sets the maximum possible size in blocks, expressed as a radius.

Maximum height at which building is alowed.

Online mode

Server checks connecting players against Minecraft accound database.
